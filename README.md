  # homebrew-tap

  Official Homebrew tap for [Spot](https://github.com/spot-nyc/spot).

  ## Install

  ```sh
  brew install spot-nyc/tap/spot
  ```

  ## Upgrade

  ```sh
  brew upgrade spot
  ```

  ## Uninstall

  ```sh
  brew uninstall spot
  ```

  ## What's in this repo?

  Only the auto-generated Homebrew formula for `spot`, in `Formula/spot.rb`. It is updated automatically by [GoReleaser](https://goreleaser.com) every time a new tag is pushed to
  [spot-nyc/spot](https://github.com/spot-nyc/spot). Please don't edit the formula by hand — changes will be overwritten on the next release.

  For source code, issues, and documentation, see the main [spot-nyc/spot](https://github.com/spot-nyc/spot) repo.
